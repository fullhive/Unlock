General Info

Unlock is a sign in template utilizing BootStrap for the design elements, along with a heavy emphasis on jQuery to access user data through ajax calls.

Pages

Unlock has five core pages:

Single Sign In

Single Sign In is a straightforward email + password sign in page.

Dual Sign In

Dual Sign In starts with a traditional sign in and then requires users to verify a 4-digit pin number before sign in is complete.

Register

Register has 4 sections of a wizard that requires the user to input their personal information, password, and 4-digit pin number. The email is checked against existing users, utilizing ajax, before allowing registration to continue.

Lock Screen

Lock Screen has a full page overlay that will fade when clicked. Afterwards, the user that is signed in has to verify their 4-digit pin to continue.

Forget Password?

Forget Password? requires the user to input their email and verify their 4-digit pin before taking action (password reset, email, etc.)

CSS

Fonts

Unlock uses Lato from Google Fonts as the primary font throughout the template.

@import url(https://fonts.googleapis.com/css?family=Lato:300,400,700,300italic,400italic);

Background

The sidebar and buttons can be styled using a markup of the BootStrap-Theme stylesheet.

  • Default ( .bg-default )
  • Primary ( .bg-primary )
  • Info ( .bg-info )
  • Success ( .bg-success )
  • Warning ( .bg-warning )
  • Danger ( .bg-danger )

JavaScript

jQuery is used extensively throughout the Unlock template. Each page links to its own .js file within assets/js. Ajax functions are called using a Get request and passing variables through the data option. For demo purposes, all user data is stored in users.js within the assets/ajax folder, and user authentication is performed after pulling in the $Users variable from ajax.
